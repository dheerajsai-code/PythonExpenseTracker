# Python & Flask Expense Tracker

A simple web application to track personal expenses, set category-based budgets, and schedule recurring expenses. It features a responsive Flask dashboard and automated testing.

## Features
* **Expense Logs:** Add, edit, delete, and view your daily expenses.
* **Monthly Trends:** Visual charts showing your spending habits over the last 6 months.
* **Budget Tracking:** Set limits for food, travel, rent, etc., and see progress bars tracking your spending.
* **Recurring Bills:** Automatically schedules daily, weekly, monthly, or yearly bills.
* **Data Export:** Export your entire expense history to a CSV file.
* **User Authentication:** Secure register/login system using hashed passwords.

## Tech Stack
* **Backend:** Python, Flask
* **Database:** SQLite3
* **Frontend:** HTML5, CSS3, Javascript
* **Testing:** Pytest

## How to Run Locally

1. Install requirements:
   ```bash
   pip install flask pytest
   ```

2. Run the application:
   ```bash
   python app.py
   ```

3. Open your browser and go to `http://127.0.0.1:5000`
